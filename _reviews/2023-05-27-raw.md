**地址**：台北市中山區樂群三路301號1樓<br>
**造訪日期**：2023.5.27<br>
**人數價位**：19884/2

星星+2，江振誠主廚aka舒酸定代言人在樂群三路aka輪胎人星光大道上開設、極具指標性而引領潮流的精緻餐廳，亦是台灣bistronomy的先驅，概念新穎而顛覆認知，食材菜色多變的同時，整體仍維持一種自然清新，或許取名RAW也是想表達直接不做作的含意？

訂位困難，指南上稱「要從網上系統成功訂位，讀者還真需要多一點耐性」，每一季菜單都會一次開放三至四個月訂位，偶有加開場次，幸運透過內線才訂到本季「Originality原創」菜單的位置。

店內裝潢氣派，兩座巨大的流線型木雕貫穿店內，分隔出廚房、吧檯、用餐區，應該是學幾何的人會喜歡的設計，其他硬體設備也相當優秀，包括後續的選刀環節算是蠻有趣。

![](https://hackmd.io/_uploads/rkZuIQiD2.jpg)

本季是一個十品的套餐，七道鹹食和三道甜點，最特別的點莫過於每組賓客可以自己在某種程度上設計自己想吃的菜單。入座後會看到桌上已經放了一個形狀殊異的「菜單」，十個方塊除了第七格「Originality」是無法更動的以外，每個方塊底下寫著這道菜的主要風味來源與口感呈現，而賓客需在六道鹹食中安排主食材的順序：蔬菜、燻蛋、A5和牛、白蘆筍、中卷、馬鈴薯；甜點部分則有三種鹹食入菜：Oscietra、小茴香、豆腐，一樣由賓客安排順序。最後會印出一份寫著Uniquely created by ()的獨特菜單。

唯一讓人疑惑的是所有文案都寫著有729種可能性，然而根據我本人多年的算術經驗應該有4320種才對。當然，亦可參考主廚原始的菜單設計。不得不說是一個極為大膽的嘗試，一季之內餐廳內部需要排練的菜色從標準的10品暴增為46品，這樣瘋狂的設計恐怕台灣目前也只有RAW能做出來。

飲品部分，本次選擇6杯的餐酒搭配，此處也同樣有customize的元素，桌上的另一個方塊是一個六面骰，每一杯pairing上桌前侍者會來詢問這杯酒希望以哪一種風格呈現，機會難得就6種各選了一次，不過考量到還有順序問題的關係，餐酒搭的部分看來也很難猜出哪一組是最好的選擇。以下開始食物本身的心得，不過因為是我自己配的所以要是我宣稱非常好吃或許有點tautological就是了。

![](https://hackmd.io/_uploads/HJk_IQiP3.jpg)

Uni / Peanut butter / ______ 我選填的是白蘆筍，用川燙、切片、煙燻的方式呈現，侍者說醬汁是模擬麻醬口味不過相似程度其實挺低的，拌勻之後是個明亮歡快的開場，和下一道料理都搭配第一杯的 “adventure” pairing，Métisse Noirs & Blanc的香檳，單喝比較冷硬，碳酸跟酸度都較高，有點礦物感跟柑橘香(侍者說有芝麻香不過我喝起來沒這感覺)，白蘆筍切片並沒有破壞食感，煙燻味在濃厚的海膽花生醬中調劑的不錯，和上一篇愛法的甜點有異曲同工之妙。

![](https://hackmd.io/_uploads/HJJdLmoP3.jpg)

Toro noisette / Mochi / ______ 這一道選的菜是和主廚一樣的和牛尖角，牛肉用松露鹽漬過，鮪魚大腹用一個極端奢侈的方式呈現，逼出焦化油後再加入燻羅勒與巴西利形成醬汁，口感來源則是麻糬和榛果。醬汁本身焦香層次就已經相當豐富，鮪魚的味道藏在最深處，和牛和麻糬的口感融合更是精妙，明明是薄切的牛肉搭上麻糬後的食感有點模擬油脂較多的肋眼部位的感覺，這一道菜也跟香檳搭得比較好。吃到此處讓人覺得果然主廚之所以是主廚，在風味上的想像就已經超越一般人太多了。某種程度上有後悔沒有完全follow主廚版菜單。

![](https://hackmd.io/_uploads/H1bd8moDn.jpg)

Maillard reaction / ______ 梅納反應這道是麵包，個人選的是中卷。提供兩種吃法，一種是傳統的奶油，另一種是特製的中卷醬料，其中還包括炸生蠔美乃滋、炸羅勒、薯絲等等會產生梅納反應的食材。醬料超級完美，上桌之後先吃一口醬料就忘記要單吃麵包，雖然奶油算不錯、也非常見的一般奶油，但是被中卷醬料輾壓，燻香和焦香交錯，一開始中卷只有提供些微口感，有些隱形，但配上第二杯 “surprise” pairing，勃艮地的自然酒 (是沒什麼聽說過的產地 Savigny les Beaune，有些硫磺氣味，酸度較高，甚至單喝有點刺) 後就會瞬間沖洗掉焦香等氣味，逼出中卷的鮮味，重點是這個動作還是commute的。醬料份量超多可以單吃，雖然是自己配的，但更讓人非常在意主廚原本設計的搭配到底有多精彩。是本日第二佳的菜色。

![](https://hackmd.io/_uploads/rJfuI7sPh.jpg)

Umami bomb / Champignons / ______ 鮮味炸彈指的是菇類、墨魚汁、豆豉等堆疊出的多種鮮味組合，這裡我選的是燻蛋，不過吃起來甚至出現在吃高檔燒肉的幻覺，燻蛋和醬汁融合的非常完美。菇類本身的味道雖有保留，不過煙燻處理後不會有原本惱人的氣味，和蛋的口感搭配也很好，而且餘韻綿長，吃完之後還是可以一直回味，雖然放在梅納反應後稍稍沒那麼驚艷，仍舊是很好的一樣菜，第三杯“unique” pairing是紅葡萄、白葡萄混釀的西班牙淡紅酒，帶些梅粉或番茄汁的氣味，但這邊酒的搭配相對普通。

![](https://hackmd.io/_uploads/BJxuIXoPn.jpg)

![](https://hackmd.io/_uploads/S1mOImoD2.jpg)

![](https://hackmd.io/_uploads/Sk7_LXiDh.jpg)

Roasted whole skate / Aubergine / ______ 前面已有桌邊服務展示將魔鬼魚刨成魚鬆，這道是魔鬼魚、巴西利、(炸)酸豆、奶油、炸麵包打成的魚漿類食物，搭配蔬菜茄泥。另有泡在苦茶油裡上桌的人蔘，沾巧克力粉弄得像是剛出土的樣子食用。比較在吃意象，也可能只是我本來就不愛這些蔬菜所以沒有特別感想，不過羽衣甘藍和甜菜根都挺好吃的，第四杯 “tradition” pairing是Ube 西班牙雪莉區的白酒，也是雪莉酒原型，往好的想是不出錯，往壞的想就是沒特色，標準合理但不突出。

![](https://hackmd.io/_uploads/B1iuUmjD2.jpg)

Memory 1997 / Foie gras / Black truffle 唯一不能更動的菜色，也是RAW的永久招牌，鴨肝做成茶碗蒸like的樣子再鋪上松露醬，不可攪拌。簡單說就是最頂級的食材，最完美的食感，無比粗暴的好吃，直逼[AJR的馬鈴薯泥](/jD5r27NmRKaJBPbarYMCUw)。

![](https://hackmd.io/_uploads/BksuLQjP3.jpg)

在主餐上桌之前有一個讓賓客選用刀具的環節，侍者會端上來一套某知名設計師替五位名廚設計的一套刀具，我們選了Andre的Concrete(特色是刀身上面雕了一個小小的台灣)和AlainDucasse的G10。

![](https://hackmd.io/_uploads/By2uLmjvn.jpg)

Pigeon / Cacao / ______ 主餐，這邊我選的是馬鈴薯，也是主廚原作。熟成三天的屏東乳鴿，表皮加了麥芽糖跟醋來烘烤，醬汁是可可醬，另外搭配調味較少的馬鈴薯跟韭蔥蕾絲餅並撒上可可粉。乳鴿皮很好吃，有點粵式燒臘的香氣，鴿肉本身沾上可可醬起初會帶點苦味，隨後浮出肉本身的鮮味，並襯出腿肉驚人的鮮嫩，馬鈴薯和蕾絲餅以口感為主，不是味道的主要來源。第五杯的 “classic” pairing是法國的紅酒，是兩種品種葡萄的混釀，酒精濃度較高，酒也是經典搭配，相對穩健。 

![](https://hackmd.io/_uploads/B1huLmjw2.jpg)

甜食部分雖然也是三道，但不是按照一般的前甜點、主甜點、茶點的順序。
Marshmallow / Goat chesse / ______ 第一道甜點就相當重口，羊乳酪冰淇淋搭配烤至完全融化成乳狀的棉花糖，鹹甜風味，呈現乳乳交融的狀態，我選的是凍豆腐，豆腐磨成粉所以一開始就揮發出完整香氣，還不錯的甜品。

![](https://hackmd.io/_uploads/B1jOL7iDn.jpg)

Artichoke / Crème brulée / ______ 第二道甜點搭配最後一杯 “timeless” pairing，是法國阿爾薩斯的Fronholz白酒，以白花香為主，也有桃子等香氣。甜點本身材料比較特別，包含辛香料跟蔬菜，主體是朝鮮薊烤布蕾搭焦糖脆片再鋪上小茴香。不過辛辣感較低，清甜爽脆為主。朝鮮薊去除了討厭的軟脆食感，反而氣味中和了焦糖烤布蕾的甜膩，茴香口感蠻特別，在焦糖脆片下可以拉長尾韻，有點像第一道白蘆筍的甜食版本。酒香如預期般是爆發力強的，其中的味道則和甜點大相徑庭卻又相得益彰。

![](https://hackmd.io/_uploads/BJ5_8Qjwh.jpg)

Lily / Sake lees / Olive oil / ______ 最後一樣我選的是魚子醬。這道以野薑花為主、百合只有在吃到的時候特別有爆發力，苦鹹和甜味的對比非常明顯。有點不適合放在最後一道，甜點部分完全可以想像主廚原本的搭配更好，凍豆腐粉和清酒粕應該是個好組合。

總而言之，是一個意圖使人同一季就二訪的菜單，創作力高而活潑的餐廳，或許不那麼適合宴客，但絕對充滿食趣和驚喜。

**環境裝潢**：90/100 缺點就是比較吵鬧...<br>
**餐點水準**：95/100 風味搭配不夠完美是我自己選擇造成的，不過飲品跟餐食的搭配沒那麼頂尖也是事實<br>
**服務品質**：85/100<br>
**整體用餐體驗評價**：92/100
